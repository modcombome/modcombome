# MODCOMBO GitHub Pages Optimized

Bộ code đã được rút gọn từ HTML WordPress xuất ra thành bản static chạy được trên GitHub Pages.

## Cách dùng

1. Tạo repository GitHub mới.
2. Upload `index.html` vào root repository.
3. Vào Settings → Pages → Deploy from branch → chọn `main` và `/root`.
4. Khi dùng domain GitHub riêng, sửa thẻ canonical nếu muốn index URL GitHub thay vì canonical về `https://modcombo.me/`.

## Đã tối ưu

- Bỏ admin bar, wp-comments, jQuery, Flatsome JS, RankMath CSS, WordPress block CSS dư thừa.
- Giữ logo, màu đen/xanh #51c97f, menu, search, card game/app/tin tức.
- Dùng HTML/CSS/JS thuần, responsive mobile.
- Giữ meta SEO, Open Graph, Twitter Card và JSON-LD.
- Sửa CSS lồng kiểu SCSS không hợp lệ trong bản xuất WordPress.
